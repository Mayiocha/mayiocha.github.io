Unidad 1 - Análisis semántico

El análisis semántico verifica que las instrucciones del programa tengan sentido lógico y respeten las reglas del lenguaje de programación. En esta unidad se estudian árboles de expresiones, comprobación de tipos, acciones semánticas y manejo de errores semánticos.

Explorar

Unidad 2 - Generación de código intermedio

La generación de código intermedio transforma las instrucciones del programa fuente en representaciones más simples y estructuradas. Esta etapa facilita la optimización y la traducción posterior a código objeto.

Explorar

Unidad 3 - Optimización

La optimización mejora la eficiencia del código generado sin modificar el comportamiento del programa. Se analizan técnicas para reducir costos de ejecución y mejorar el rendimiento general.

Explorar

Unidad 4 - Generación de código objeto

La generación de código objeto transforma el código intermedio en instrucciones cercanas al lenguaje máquina, considerando registros, memoria y arquitectura del sistema.
